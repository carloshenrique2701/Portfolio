Como converter um texto para binário

Para converter um texto para binário, basta digitar o texto na caixa de texto e clicar no botão "Converter para binário".

Lógica utilizada no código:

  1. Obtivemos o código ASCII do caractere: Cada caractere em um texto possui um código numérico único chamado código ASCII. Por exemplo, o caractere "A" possui o código ASCII 65.

  2. Convertemos o código ASCII para binário: Convertemos o código ASCII de decimal para binário. O código 65 em binário é 1000001.

  3. Padronizamos a representação binária: Padronizamos a representação binária para ter 8 bits (adicionando zeros à esquerda, se necessário). O código binário 1000001 se torna 01000001.

  4. Repitimos o processo para todos os caracteres: Repitimos os passos acima para todos os caracteres do texto. Por exemplo, para o texto "AB", você teria:
    • A: 01000001
    • B: 01000010

  5. Juntamos todas as representações binárias: Juntamos todas as representações binárias com um espaço entre elas. Para "AB", o resultado seria 01000001 01000010.

Para converter de binário para texto, siga o processo inverso:

  1. Divida a string binária em blocos de 8 bits.
  2. Converta cada bloco de 8 bits de binário para decimal (código ASCII).
  3. Obtenha o caractere correspondente ao código ASCII.
  4. Junte todos os caracteres para formar o texto original.
