Estoque de produtos

Desenvolvi um sistema de estoque em Java, onde o usuário pode cadastrar, editar e excluir produtos, além de realizar o gerenciamento de outros usuários (ADM). O sistema utiliza um banco de dados MySQL para armazenar as informações localmente. Ao criar um controle de acesso baseado em roles, o sistema garante que apenas os usuários com permissões adequadas tenham acesso ao sistema.

Características

  • Sistema de Autenticação Robusto - Login seguro com controle de permissões
  • Gestão de Usuários - Suporte a múltiplos tipos de usuário (Administrador/Gerente)
  • Controle de Produtos - CRUD completo com validações
  • Gestão de Fornecedores - Cadastro e vinculação com produtos
  • Interface Intuitiva - Design responsivo com abas organizadas
  • Edição em Tempo Real - Modificação direta na tabela de dados
  • Ordenação Automática - Listas organizadas por ID crescente
  • Interface Personalizável - Campos de input com tamanhos ajustáveis

Observação Importante: O código do sistema foi feito 80% em IA, o intuito desse projeto era, além de entender como funcionava uma estrutura MVC, era a criação de diagramas UML para documentação(em teor acadêmico).
